With the App Identity and Access Adapter, you can use any OIDC compliant identity provider to control authentication and authorization policies in all environments, including frontend and backend applications, without any change to your code or the need to redeploy your application. Refer to the App Identity and Access Adapter docs for more information and sample configurations and policies.
